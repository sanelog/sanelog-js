# sanelog

> Sane log aggregation. Logging that doesn't drive you mad.

**⚠️ This package is a placeholder.** It reserves the `sanelog` name on npm
for the official [sanelog.com](https://sanelog.com) SDK, which is currently
in development.

## What is sanelog?

sanelog is a log aggregation platform — collect, search, and live-tail logs
from all your apps and servers in one place, through an interface designed
to stay out of your way.

## What will this package become?

The official Node.js SDK and CLI for sanelog:

- 📡 Ship logs from your Node apps with a couple of lines of code
- 🔍 Query and tail your logs from the terminal
- ⚙️ Manage sources, alerts, and retention from CI

## Current status

npx sanelog
